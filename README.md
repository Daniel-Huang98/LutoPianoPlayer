# Installation

> npm i

# Run

> npm run-script run

# Test

> npm test

# Data Type

<p>Refer to input.json for reference</p>
<code>
[
    {
        "key": string,
        "sleep_time": number
    }
]
</code>

<ul>
<li><p>key: determines which key to press. Refer below for key-string equivalents.</p></li>
<li><p>sleep_time: determines how long to hold the key for in milliseconds</p></li>
</ul>
